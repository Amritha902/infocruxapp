# **App Name**: Market Pulse AI

## Core Features:

- User Authentication: Secure user authentication using Email/Phone OTP with JWT session management.
- Broker API Integration: OAuth-based read-only integration with broker APIs (e.g., Sharekhan, Zerodha) to fetch holdings and order history.
- Portfolio Dashboard: Comprehensive dashboard displaying holdings summary, P&L, sector allocation, and risk exposure.
- Watchlist Module: Custom watchlists with real-time price updates and quick anomaly badge display.
- Stock Detail Page: Live price chart, OHLCV data, volume data, recent announcements, and ML Risk Indicator (Low/Medium/High) with explanation.
- Announcement Intelligence Panel: Corporate announcement feed with impact severity score, timeline visualization, and before/after price chart overlay.
- AI-Powered Anomaly Detection: ML-based engine that uses Event window modeling, abnormal return detection, volume spike detection, liquidity anomaly detection. ## Style Guidelines:

- Primary color: Deep Blue (#1A237E) to evoke trust and stability, inspired by traditional finance but with a modern twist.
- Background color: Light Gray (#F5F5F5), almost white, to provide a clean and uncluttered backdrop.
- Accent color: Teal (#008080) for interactive elements and highlights, providing a fresh contrast to the blue.
- Headline font: 'Poppins', sans-serif, for a precise and contemporary feel, well-suited for headlines and short amounts of text. For longer text, use 'PT Sans'.
- Body font: 'PT Sans', humanist sans-serif, combining a modern look with a little warmth and personality, for body text.
- Minimalist, line-style icons for key features, ensuring a clean and uncluttered interface.
- Mobile-first, high information density layout inspired by Sharekhan, Zerodha Kite, and Upstox.
- Subtle transitions and animations for a smooth and engaging user experience.
